### Stack Tecnológico

| Capa              | Tecnología       | Descripción                   |
| ----------------- | ---------------- | ----------------------------- |
| **Frontend**      | React 18 + Vite  | Aplicación SPA moderna        |
| **Estilos**       | TailwindCSS      | Framework CSS utilitario      |
| **Backend**       | NestJS (Node.js) | Framework progresivo de Node  |
| **ORM**           | TypeORM          | Mapeo objeto-relacional       |
| **Base de Datos** | MySQL 8.0        | Sistema de gestión relacional |
| **Autenticación** | JWT              | Tokens web JSON               |
| **Contenedores**  | Docker           | Virtualización ligera         |
